# Vertical City Simulation (C Language)

A professional simulation of a vertical city building system using the **C language** and **Stack data structure**.

###  Key Features
- **Build (Push):** Add new building layers with weight constraints.
- **Demolish (Pop):** Remove the top-most layer following LIFO logic.
- **Validation:** Ensures a Power Plant is built first and monitors 800-unit weight limit.

### Technical Implementation
- Structs for City and Layer management.
- Dynamic power consumption logic.
- Standard C input/output for console interaction.
